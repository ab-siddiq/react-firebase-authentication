# Firebase authentication with react

## What is used in this project?

- `Firebase` for user login authentication
    - `Gmail Sing In`
    - `Github Sign In`
    - `Facebook Sign In`
    - `Email Sign In`
    - `Email verification`
    - `Forgot password`
- `React.js` for building this project
- `tailwind css` for ui design
- `React Router` for navigating pages
